FAQ

How can you find the parent or children of an element?

element.parentElement: Gets the parent element.
element.children: Returns a live HTMLCollection of an element's child elements.
element.firstChild / element.lastChild: Get the first/last child node of the element.

What is the difference between innerHTML and textContent?

innerHTML returns or sets the HTML content of an element, including any HTML tags.
textContent returns or sets only the text within an element, ignoring any HTML tags.

What are DOM nodes and elements?

A DOM node represents any object in the DOM tree
A DOM element is a specific type of node representing an HTML element.

How can you stop event propagation?

stop event propagation by using the stopPropagation() method. This prevents the event from bubbling up to parent elements.
