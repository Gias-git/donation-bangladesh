FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page as a tree of objects and nodes, allowing programs to interact with and manipulate the page's content, structure, and style.

How do you select an element from the DOM?

getElementById()
getElementsByClassName()
querySelector()
querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the context of the DOM (Document Object Model) where you take advantage of event bubbling to manage events more efficiently. Instead of attaching event listeners to individual child elements, you can attach a single event listener to a parent element that listens for events triggered by its children. This listener can then handle the events for all the child elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, you can access the element using methods like getElementById or querySelector, then use the setAttribute method to change attributes and the style property to modify inline CSS styles directly on the element; for example, element.setAttribute('href', 'new-link') to change an anchor tag's link, or element.style.color = 'red' to set the text color of an element to red.
